# React Quick starter - Typescript
A drop-in replacement for CRA (Create React App) with simple yet advance setup.

### Equipped with

- Redux ( RTK Query + Redux Toolkit )
- React Router Dom
- Typescript
- TailwindCSS
- MantineUI
- Prettier
- EsLint
- Vite
